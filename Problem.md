# Acer Swift 3 筆電 無線網卡壞掉
- 狀況
    - Acer Swift 3 筆電 無線網卡壞掉
    - 設定選單中找不到 WiFi
- 解法
    1. 實體網路線轉 轉接器 Hub / DONGLE 轉 USB轉 USB
    2. 透過 iPhone: 用 USB 連接電腦後，手機開啟網路分享，開啟 iTunes，可看到類似乙太網路的圖示顯示於狀態列中。
    3. 買 USB 無線網卡 (TP-Link 之類)
